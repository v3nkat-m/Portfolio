CONDUIT

A full stack blog website with features like authentication and membership using React and NodeJS

Demo

Project Overview

  • Backend Development: Developed the backend using Node.js, Express, and MongoDB for efficient server-side operations and data management.
  • Frontend Development: Utilized React, CSS, and React-Router for building reusable components, stylish designs, and seamless client-side routing.
  • React Hooks: Leveraged React hooks like useContext, useState, useRef, and useEffect for code efficiency and development simplification.
  • Web Requests: Used Axios for reliable communication between frontend and backend, ensuring efficient handling of HTTP requests.
  • Authentication and Sessions: Implemented secure user authentication strategies with PassportJS, including Google OAuth, and session management with Redis.
  • File Handling: Integrated Cloudinary and Multer for efficient management of user-uploaded images.
  • API Endpoints and Features: Implemented various endpoints for authentication, CRUD operations, likes, follows, and nested comments.
  • Membership and Payment: Developed a membership page with Stripe API integration for seamless payment functionality.
  • Testing: Used Postman for testing API endpoints.
